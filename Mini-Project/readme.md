## 📝 Description
This project was developed as the final capstone for the **JSONPowerDB Introduction Course**. It features a "Smart Form" logic that integrates directly with a JPDB backend to provide seamless data entry, real-time validation, and automated record retrieval.

## ⚡ Why JSONPowerDB?
I chose JSONPowerDB for this project because:
* It is **really fast** at saving and finding data (Real-time).
* It doesn't need a complicated setup like traditional SQL databases.
* It works directly with JavaScript via REST APIs.
* It handles "Schema-less" data, making development agile.

## 🚀 Key Features
* **Smart ID Check**: Instant database lookup on Roll No / Employee ID entry.
* **Auto-Fill**: Automatically populates existing records for editing.
* **Dynamic State Management**: Locks/Unlocks fields and buttons based on record existence.
* **Input Validation**: Ensures data integrity before submission.
* **Reset Functionality**: One-click form clearing and focus management.
